Sustainable.css

A sustainable and good-enough CSS boilerplate that:

  1. Leverages web browsers built-in aesthetics.
  2. It embraces semantic HTML.
  3. It focuses in basic styling, not layout.
  4. Is built with branding in mind so you can color pallette your site.
  5. Is built with sustainability in mind.

It is 2023, Web Browers are amazing at rendering and handling natively light-dark schemes. Let's built on that.

Yet another minimalistic CSS Framework?

Yes.

I built sustainable.css because I usually do lots of simple websites that require two basic things:

  1. Some kind of centering
  2. Customizing color pallettes.
  3. Support for dark/light schemes
I built this framework to help me with that. I hope it helps you too.

Is it for me?

Sustainable.css aims to be good enough at styling plain HTML. Perfect for MVPs and sites built on HATEOAS.

It does not provide styling that you will probably change anyways (like headers font size and margins) nor layout system (it's 2023, you should be leveraging modern CSS).

Sustainability?

With sustainability I refer to several things:

  1. It is just a few KB, Sustainable.css does not require lot of bandwith and energy to work.
  2. It will work as long as HTML and CSS work, with extremely low maintenaince.
  3. It is not hosted nor packaged with expensive pipelines or infrastructure.
  4. Web browsers spends lots of money on styling and rendering features. I build on top of that.

A Good Enough color pallette

Since I'm trying to delegate as much as possible to the browser defaults, I just choose a color that goes good with Black & White (Which Colors Look Good on Black and White?). This way I can leverage the browser's light-dark scheme, the user's preferences and I make sure that colors I choose are accesible.

I might extend this as I explore and understand better minimalistic ideas to add color to a website.

Customization

Just add your own css file and override/extend whatever you want.

So far only 2 variables are exposed:

  • --max-width to control the width of the body
  • --theme-color to control the color of your theme.
